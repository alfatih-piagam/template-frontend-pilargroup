## 🔧 Cara Instalasi dan Menjalankan Proyek

Ikuti langkah-langkah berikut untuk menjalankan proyek di lingkungan lokal Anda:

1. **Clone repositori ini:**
   ```bash
   git clone https://github.com
   ```

2. **Masuk ke direktori proyek:**
   ```bash
   cd nama-repo
   ```

3. **Instal dependensi:**
   ```bash
   npm install
   # atau jika menggunakan yarn/pnpm
   yarn install / pnpm install
   ```

4. **Jalankan server pengembangan:**
   ```bash
   npm run dev
   ```

5. **Buka di browser:**
   Aplikasi akan berjalan di alamat `http://localhost:5173` (atau port yang tertera di terminal Anda).

## 📦 Skrip yang Tersedia

Di dalam proyek ini, Anda dapat menjalankan perintah berikut:

- `npm run dev` - Menjalankan aplikasi dalam mode pengembangan.
- `npm run build` - Mengompilasi aplikasi untuk siap diproduksi ke folder `dist`.
- `npm run preview` - Meninjau hasil build produksi secara lokal.
- `npm run lint` - Memeriksa kesalahan kode menggunakan ESLint.
